# FolderSync 用户使用手册

欢迎使用 **FolderSync** —— 一个安全、快速、跨平台的点对点（P2P）文件同步工具。

## 1. 快速入门

### 安装与启动
- **MacOS**: 运行 `FolderSync.App.app`。首次运行时，系统可能会提示是否允许网络访问，请选择“允许”以确保 P2P 功能正常。
- **Windows**: 运行 `FolderSync.App.exe`。

### 第一次同步
1. **启动应用**：打开 FolderSync 仪表盘。
2. **添加文件夹**：点击“添加文件夹”按钮，选择您想要同步的本地目录。
3. **发现节点**：只要在同一个局域网（LAN）内，您的其他电脑也会自动出现在“发现的节点”列表中。
4. **开始同步**：应用会自动检测文件更改并开始同步。

## 2. 核心功能

### P2P 发现与设备管理
FolderSync 不通过云端，而是直接在您的设备间传输。
- **自动发现**：同一网络下的设备会自动显示。
- **设备配对**：在“协同设备”标签页，您可以管理所有已知设备。点击“信任”即可与新设备建立安全连接。

### 版本控制与备注
应用会自动保留文件的历史版本（默认 10 个）。
- **版本历史**：在“版本历史”标签页查看所有文件的备份。
- **版本备注**：您可以为重要的备份添加“备注”，方便日后识别。
- **版本对比**：右键点击文件（或在版本页面）可以查看文本文件的代码差异。

### 安全与隐私
- **应用锁 (App Lock)**：在“设置”中开启应用锁并设置 4-6 位 PIN 码，保护您的配置不被他人随意更改。
- **异动监测**：如果短时间内检测到大量删除操作（可能是误操作或异常），应用会发出安全警报。
- **数据流加密**：所有文件传输均经过 AES-256 加密。

### 过滤与计划
- **过滤器**：支持 JSON 格式的规则，可排除 `.git`, `node_modules` 等无关文件夹。
- **同步计划**：在“设置”中可以配置允许同步的小时数，例如仅在夜间 `00:00 - 06:00` 同步。

## 3. 界面指南
- **主页**：查看当前文件夹同步状态和节点概况。
- **协同设备**：管理信任关系，查看远程节点在线状态。
- **版本历史**：浏览历史归档并管理版本备注。
- **同步记录**：查看所有文件的历史操作日志（新增、修改、删除）。
- **文件冲突**：处理由于两地同时修改导致的冲突文件。
- **设置**：配置设备名、同步规则、语言（中/英）及应用锁。

## 4. 常见问题 (FAQ)

**Q: 如何切换语言？**
A: 在“设置”页面的语言下拉框中选择“English”或“简体中文”，界面将立即切换。

**Q: 为什么我有冲突文件？**
A: 当两个设备同时修改了同一个文件，为了防止覆盖，FolderSync 会保留双方的版本，请在“文件冲突”页进行手动决策。
